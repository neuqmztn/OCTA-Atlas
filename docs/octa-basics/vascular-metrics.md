# 血管指标：VD、VLD、FAZ、FD、VSD、EN、JN

血管指标用于量化 OCTA 图像中的血管密度、形态、拓扑和区域特征，是连接图像分析与临床解释的重要桥梁。

| 指标 | 含义 | 备注 |
| --- | --- | --- |
| VD | Vessel Density | 血管像素面积比例 |
| VLD | Vessel Length Density | 骨架长度密度 |
| FAZ | Foveal Avascular Zone | 黄斑中心无血管区 |
| FD | Fractal Dimension | 分形维数 |
| VSD | Vessel Skeleton Density | 血管骨架密度 |
| EN | Endpoints Number | 端点数量 |
| JN | Junctions Number | 分叉点数量 |

## 使用提醒

不同论文对预处理、阈值化、骨架化和区域划分的定义可能不同，比较结果时需要先统一计算流程。
