# OCTA Basics｜OCTA 基础知识

OCTA Basics 是 OCTA-Atlas 中面向新同学的基础知识入口，主要用于建立从 **OCT 结构成像 → OCTA 血流成像 → OAC 组织光学衰减成像 → 血管分层 → 定量指标 → 常见伪影** 的完整认知链路。很多 OCTA 论文读不懂，并不一定是模型结构复杂，而是还没有把成像原理、医学区域、指标定义和图像质量问题联系起来。

本板块既适用于视网膜 OCTA 研究，也兼顾实验室已有的小动物脑缺血 OCTA/OAC 数据。前者更关注视网膜血管层次、FAZ、血管密度、疾病分类与分割；后者更关注脑缺血后的灌注下降、脑水肿形成、OAC 变化和灌注—水肿耦合分析。

## 推荐阅读顺序

建议新同学按下面顺序阅读：

1. **OCT 基础**：先理解 OCT 为什么能提供深度分辨的组织结构信息。
2. **OCTA 基础**：再理解 OCTA 如何利用重复扫描中的运动对比提取血流灌注信息。
3. **视网膜分层**：学习 SVP、ICP、DVP、CC 等常见血管层次及其在论文中的含义。
4. **血管指标**：理解 VD、VLD、FAZ、FD、VSD、EN、JN、VPD、OAC 等指标如何计算、如何解释。
5. **常见伪影**：最后了解运动伪影、投影伪影、分割错误和低信号伪影对模型与指标的影响。

## 学习目标

完成本板块学习后，应能回答以下问题：

- OCT、OCTA 和 OAC 分别看什么，它们之间是什么关系？
- OCTA 中的血流信号为什么依赖重复扫描和运动对比？
- en face OCTA 图像、B-scan、C-scan、volume 之间有什么区别？
- SVP、ICP、DVP、CC 等层次的生理意义和图像表现有什么不同？
- 血管密度、骨架长度、分形维数、端点/分叉点数量分别反映血管网络的哪些方面？
- 脑缺血 OCTA/OAC 数据中，VPD、OAC、edema area、LCBP 等指标如何共同描述“血流下降—水肿形成—周围灌注恶化”的过程？
- 常见伪影会怎样影响血管分割、指标统计、疾病分类和图像复原任务？

## 与课题组研究方向的关系

OCTA-Atlas 的基础知识部分不是孤立的科普页面，而是服务于后续课题设计：

- **疾病分类**：需要理解不同血管层、不同指标与病理变化之间的关系。
- **血管分割**：需要知道哪些血管结构重要，哪些伪影会干扰标签和预测。
- **伪影去除**：需要明确图像增强是否真正保护了血管结构和下游指标。
- **血管拓扑建模**：需要把像素级血管图转换为端点、分叉点、血管段和网络结构。
- **多模态建模**：需要理解 OCTA 侧的灌注信息与 OAC 侧的组织光学/水肿信息如何互补。
- **纵向预测**：需要将单帧图像指标扩展到时间序列变化，例如基线百分比、损伤扩张速度和局部区域风险。
