# 视网膜分层：SVP、DVP、ICP 等

OCTA 研究常按血管层次分析视网膜微循环。不同层次的分割边界、投影伪影和病理表现会影响后续模型设计与指标解释。

## 常见层次

- SVP：Superficial Vascular Plexus
- ICP：Intermediate Capillary Plexus
- DVP：Deep Vascular Plexus
- CC：Choriocapillaris

## 记录建议

阅读论文时，注意作者使用的分层定义、设备来源和导出方式是否一致。
